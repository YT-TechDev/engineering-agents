# Public OSS Hygiene

This repository is public-facing documentation. Keep all content safe to publish.

## Do Not Include

- ChatGPT shared links or private AI session links
- Codex task URLs
- Claude session URLs
- Local machine paths
- Generated-by signatures
- Private tool metadata
- Internal-only notes, credentials, tokens, or unreleased business context

## Preferred Practice

- Use generic examples and placeholders.
- Keep instructions tool-neutral unless a file is explicitly tool-specific.
- Review changed files before opening a pull request.
- Remove process notes that are only useful during drafting.
