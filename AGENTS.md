# Repository Agent Instructions

This repository is docs-only. It contains role-based AI agent files, workflow templates, and supporting documentation for engineering-related work.

## Maintenance Rules

- Prefer small, focused documentation changes.
- Do not add application code, package manager files, dependencies, build systems, or generated files.
- Preserve public OSS hygiene in every file and pull request.
- Avoid broad rewrites unless the task explicitly requires them.
- Use targeted reads before expanding scope to additional files.
- Validate Markdown headings, links, formatting, and expected file structure.
- Never push directly to `main`; use a branch and open a pull request.

## Public OSS Hygiene

Do not include ChatGPT shared links, Codex task URLs, Claude session URLs, private AI session links, local machine paths, generated-by signatures, private tool metadata, or internal-only notes.
